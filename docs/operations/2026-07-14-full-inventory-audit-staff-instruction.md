# 2026-07-14 전체 재고 실사 업무 지시

아래 첫 번째 메시지는 오늘 저녁 담당 직원에게 그대로 전달합니다. 두 번째 체크리스트는 제출 후 사장님 검토용입니다.

## 직원 전달 메시지

[오늘 저녁 전체 재고 실사 업무]

담당: 1명

시간: 3시간

범위: 매장 전체 활성 장비 전부 — 수량 제한 없이 전체를 완료해 주세요.

1. 시작 전에 반출·반납·자리 이동을 포함한 모든 장비 이동이 완전히 멈췄는지 확인해 주세요.
2. 헤이빌리에서 `재고 > 전체 재고 실사 시작`을 누르고, 매장 동선대로 끝까지 한 바퀴 돌며 모든 장비를 확인해 주세요.
3. 입력 항목은 줄이거나 생략하지 않습니다. 품목마다 아래 내용을 모두 기록해 주세요.
   - 현재 위치
   - 정상 수량
   - 정비중 수량
   - 파손·격리 수량
   - 상태 미확인 수량
   - 누락 구성품
   - 특이사항
   - 필요한 사진
   - 식별 상태: 확실 / 식별 불확실 / 목록에 없음
4. 네 가지 수량은 서로 겹치지 않게 나눠 주세요. 합계가 현장에서 실제로 발견한 수량이어야 합니다.
5. 같은 품목이 여러 위치에 있으면 하나로 합치지 말고 위치별로 각각 추가해 주세요.
6. 비슷한 모델이나 구성품 때문에 애매해도 작업을 멈추거나 사장님께 묻지 마세요.
   - 목록에는 있지만 확신이 없으면 `식별 불확실`
   - 목록에서 찾을 수 없으면 `목록에 없음`
   - 장비 전체와 모델명·시리얼이 확인되도록 사진과 메모를 남겨 주세요.
7. 파손·격리·정비중·구성품 누락이 있으면 문제 부분이 확인되는 사진과 구체적인 메모를 남겨 주세요.
8. 수량 `0`은 해당 품목의 보관 위치와 케이스 등을 실제로 확인했는데도 하나도 없을 때만 입력합니다. 아직 확인하지 않은 품목은 `0`이 아니라 `미계수`입니다.
9. 새로고침되거나 앱이 중단되면 `전체 재고 실사 이어하기`로 계속 진행해 주세요.
10. 전체 매장을 모두 확인한 뒤 `미전송 텍스트 0건`, `미전송 사진 0건`인지 확인하고 제출해 주세요. 남은 전송이 있으면 제출하지 말고 재시도해 주세요.
11. `제출 완료 · 승인 대기`가 표시된 뒤 장비 이동을 다시 시작해 주세요.

제출만으로 기존 재고 원장은 바뀌지 않습니다. 사장님이 검토하고 최종 승인한 뒤에만 반영됩니다. 실사 중에는 기존 재고 화면에서 별도로 수량을 수정하지 마세요.

## 사장님 제출 후 확인 체크리스트

- [ ] 세션이 `제출됨 · 승인 대기`인지 확인
- [ ] 미계수 품목이 남지 않았는지 확인
- [ ] 명시적 0개 품목을 우선 확인
- [ ] `식별 불확실`·`목록에 없음` 항목의 사진과 모델명·시리얼 확인
- [ ] 파손·격리·정비중·상태 미확인·구성품 누락 항목 확인
- [ ] 같은 품목의 위치별 관측이 올바르게 합산됐는지 확인
- [ ] 대여중 장비 자동 매칭과 애매한 대여 품목 확인
- [ ] 외부 수리·직원 보관 등 매장 밖 수량을 근거 확인 후 추가
- [ ] 각 품목을 `실사 반영 / 기존 원장 유지 / 재실사 요청`으로 결정
- [ ] 제출 뒤 원장이 변경되어 충돌한 품목은 다시 결정
- [ ] 모든 예외가 해소된 뒤 최종 승인
- [ ] 승인 후 재고 원장·감사 이력·마지막 확인 정보를 확인
- [ ] 장비마스터 시트 반영 상태가 완료인지 확인하고, 실패 시 재시도

# 빌리지 타임라인 v2 — 디자인 스펙

## 요약

vis-timeline 라이브러리를 완전 제거하고 순수 HTML/CSS/JS로 노션 수준의 타임라인 뷰를 새로 구축한다. 렌탈샵 특화 기능(재고 부족 경고, 오늘 반출/반납 하이라이트, 매출 미니 차트, 빠른 상태 변경, 인라인 예약 추가)을 포함한다. 반응형 단일 파일로 데스크탑/모바일 통합.

## 배경

- 현재: `docs/timeline.html` (vis-timeline 기반, ~885줄) + `timelineMobile.html` (별도 모바일)
- 문제: vis.js 기본 스타일에 `!important` 30개+ 오버라이드로도 "라이브러리 느낌" 탈피 불가. 기능도 부족.
- 목표: 노션 타임라인을 뛰어넘는 렌탈 전용 타임라인

## 기술 스택

- **순수 HTML/CSS/JS** — 외부 라이브러리 없음
- 가로 스크롤: 네이티브 scroll (하드웨어 가속)
- 줌: CSS Grid 컬럼 너비 조절
- 드래그/리사이즈: Pointer Events API
- 그룹 접기: CSS transition
- 배포: GitHub Pages (`docs/timeline.html` 교체)
- API: 기존 GAS endpoint 그대로 (`?key=village2026&action=timeline`)

├─────────────────────────────────────────────────────┤
│ 필터바: [대기 12] [반출중 8] [반납완료 23] ... 통계  │
├──────────┬──────────────────────────────────────────┤
│ 라벨패널 │ 날짜 헤더 (sticky)                        │
│          ├──────────────────────────────────────────┤
│ ▼ 세트A  │ ██ 바 ██    ██ 바 ██                     │
│   장비1  │   김영수        박지은                     │
│   장비2  │ ██ 바 ██                                  │
│ ▶ 세트B  │ (접힌 상태)                               │

## 컴포넌트 상세

### 1. 헤더

| 요소 | 설명 |
|------|------|
| 타이틀 | "빌리지 스케줄" 고정 텍스트 |
| 뷰 탭 | 타임라인 / 캘린더 / 리스트 (v2에서는 타임라인만 활성, 나머지는 placeholder) |
| 그룹핑 드롭다운 | 세트별(기본) / 고객별 / 상태별 — 선택 시 왼쪽 라벨 + 그리드 재그룹핑 |
| 검색 | 세트명, 고객명으로 필터링. 150ms debounce |
| 필터 버튼 | 상태 필터 (진행중만 보기 등) |
| 오늘 버튼 | 오늘 날짜로 스크롤 |
| +예약 버튼 | 새 예약 폼 팝업 |

**모바일**: 검색은 아이콘 탭 시 확장, 뷰탭은 드롭다운으로 축소

### 2. 필터 바

- 상태별 칩: 대기 / 반출중 / 반납완료 / 취소 — 클릭으로 토글(숨기기/보이기)
- 통계: `N개 세트 · M건 · 오늘 반출 X 반납 Y`

### 3. 라벨 패널 (왼쪽)

- 너비: 220px (데스크탑), 모바일에서는 축소 또는 숨김
- 그룹 헤더: ▼/▶ 토글 + 세트명 + 건수 뱃지
- 하위 행: 장비명 (들여쓰기)
- 스크롤: 그리드 패널과 수직 동기화

### 4. 그리드 패널 (오른쪽)

- **날짜 헤더** (sticky top): 월/일/요일 표시. 주말 배경 음영, 오늘 파란 하이라이트
- **그리드 셀**: 날짜 컬럼 × 장비 행. 주말/오늘 배경 구분
- **오늘 라인**: 빨간 세로선, "오늘" 라벨
- **가로 스크롤**: 네이티브 overflow-x scroll
- **라벨/그리드 수직 동기화**: 한쪽 스크롤 시 다른 쪽도 동기화

### 5. 타임라인 바

**텍스트 형식**: `세트명 / 고객명 / HH:MM-HH:MM(N회차)`
- overflow 시 `text-overflow: ellipsis`
- 호버 시 툴팁으로 전체 정보 표시

**상태별 색상**:
| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 대기 | #D3E5EF | #24609C |
| 반출중 | #FDECC8 | #93640D |
| 반납완료 | #DBEDDB | #2B6B2B |
| 취소 | #E8E8E5 | #787774 (취소선) |
| 기타 | #E8DEEE | #6940A5 |

**인터랙션**:
- 호버: 미세한 그림자 + translateY(-1px)
- 클릭: 상세 패널 열기
- 드래그: 날짜 이동 → API 호출 (`action=updateTime`)
- 리사이즈: 양쪽 핸들 드래그로 기간 조절 → API 호출
- 우클릭/롱프레스: 컨텍스트 메뉴

### 6. 컨텍스트 메뉴 (우클릭/롱프레스)

| 항목 | 동작 |
|------|------|
| 반출 처리 | 상태를 "반출중"으로 변경 → API |
| 반납 완료 | 상태를 "반납완료"로 변경 → API |
| 상세 보기 | 상세 패널 열기 |
| 계약서 열기 | 계약서 링크 새 탭 |
| 알림톡 발송 | 알림톡 API 호출 |
| 취소 | 상태를 "취소"로 변경 → API |

**모바일**: 롱프레스(500ms) 시 바텀시트 형태로 표시

### 7. 재고 부족 경고

- 장비마스터에서 장비별 재고 수량 조회
- 동일 장비 + 겹치는 기간의 예약 건수가 재고 수량 초과 시에만 경고
- 표시: 빨간 outline + 펄스 애니메이션
- API에서 재고 정보를 함께 내려주거나, 별도 호출 필요 (기존 API 확장)

### 8. 오늘 반출/반납 하이라이트

- 오늘 반출 예정: 바 왼쪽에 📤 아이콘
- 오늘 반납 예정: 바 오른쪽에 📥 아이콘
- 필터바 통계에 "오늘 반출 N 반납 M" 표시

### 9. 매출 미니 차트

- 타임라인 하단 고정 영역 (높이 48px)
- 날짜 컬럼과 1:1 대응하는 막대그래프
- 데이터: 타임라인 아이템의 단가 합산 (일별)
- 호버 시 금액 표시
- 가로 스크롤: 그리드 패널과 동기화

### 10. 타임라인에서 예약 추가

- 빈 영역 드래그: 반출일~반납일 범위 선택
- 드래그 완료 시: 간단한 예약 폼 팝업 (고객명, 세트 선택 등)
- 폼 제출 → 기존 확인요청 API로 등록
- 모바일: 빈 영역 탭 시 날짜 선택 → 폼

### 11. 줌 컨트롤

- 우하단 플로팅: +/−/현재 스케일 표시
- 줌 레벨: 일(기본) / 주 / 월
- 줌 시 CSS 변수 `--col-w` 조절 (일: 120px, 주: 200px per week, 월: 160px per month)
- 핀치 줌 지원 (모바일)

### 12. 상세 패널

- 데스크탑: 오른쪽 슬라이드인 패널 (320px)
- 모바일: 바텀시트 (max-height: 60vh)
- 내용: 거래ID, 고객명, 연락처, 세트명, 장비명, 수량, 반출/반납 일시, 상태, 단가

### 13. 그룹핑 모드

| 모드 | 그룹 기준 | 하위 행 |
|------|-----------|---------|
| 세트별 (기본) | 세트명 | 개별 장비 |
| 고객별 | 고객명 | 예약 건별 |
| 상태별 | 대기/반출중/반납완료/취소 | 예약 건별 |

그룹핑 전환 시 동일 데이터를 재그룹핑만 하면 됨 (API 재호출 불필요)

## API 변경 사항

### 기존 유지
- `?action=timeline` — 그룹/아이템 데이터 (그대로)
- `?action=updateTime` — 드래그 후 시간 업데이트 (그대로)

### 신규 필요
- `?action=timeline` 응답에 추가 필드:
  - 각 아이템에 `회차` (N회차) 정보 추가
  - 각 아이템에 `반출시간`, `반납시간` (HH:MM 형식) 별도 필드 추가
- `?action=timeline` 응답에 장비별 재고 수량(`stock`) 필드 추가 — 프론트에서 겹치는 기간 계산하여 부족 여부 판별 (별도 API 불필요)
- `?action=updateStatus` (신규): 상태 변경 (대기→반출중→반납완료 등)

## 파일 변경 계획

| 파일 | 변경 |
|------|------|
| `docs/timeline.html` | 전체 교체 (새 타임라인 v2) |
| `timelineMobile.html` | 폐기 (반응형으로 통합) |
| `timeline.html` (GAS용) | `docs/timeline.html`로 리다이렉트 또는 동일 교체 |

| `sheetAPI.js` | `updateStatus` 액션 추가 |

## 스타일 가이드

- 폰트: 시스템 폰트 스택 (`-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Noto Sans KR', sans-serif`)
- 색상: 노션 컬러 팔레트 기반 (목업의 CSS 변수 참조)
- 라운딩: 6px (바: 5px)
- 그림자: 최소화 — 호버/플로팅 요소에만
- 애니메이션: 150ms ease 기본, 줌/패닝은 400ms easeInOutQuad

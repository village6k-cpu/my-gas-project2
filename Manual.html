기본은 AI 자동. 수동 시 동행모드.

✋ 수동 입력 필요 시 → 메뉴 📋 빌리지 스케줄 → 🎓 동행 모드 열기. AI가 단계별로 코칭함.
1

기본 정보 입력

B~E 일시 F 장비/세트 G 수량 K~L 예약자

A열 요청ID 자동생성 / 세트는 구성품 자동 펼침

2

가용 확인

H = 확인

✅ 가용 / ⚠ 부족 / ❌ 불가 결과 확인

⚠ 결과 확인 후 승인! 한 번 보내면 취소 불가.

예약 등록 완료

N = 등록

계약마스터·스케줄상세·개고생2.0·계약서·완료알림톡 모두 자동.

예약 변경 / 취소 / 추가

이미 등록된 건 — 확인요청 거치지 말고 직접 수정이 편함.

✋ 직접 수정 시 빌리지 스케줄(타임라인)에서 가용여부 수동 체크 필수.

📅 일정 변경

계약마스터 E~I열 직접 수정
→ 스케줄상세 / 거래내역 / 계약서 자동 동기화 (3초 디바운스)

➕ 장비 추가

A안 오늘 일정 웹앱 카드 → + 장비추가
B안 스케줄상세 빈 행 B열에 거래ID 입력 → 나머지 자동 / D열에 장비 선택
→ 계약서 자동 재생성

➖ 장비 삭제

A안 오늘 일정 웹앱 카드 → × 클릭
B안 스케줄상세 해당 행 우클릭 → 행 삭제

🚫 계약 취소

계약마스터 J = 취소 선택
→ 스케줄상세 / 거래내역 자동 삭제 + 행 빨강+취소선 표시
💡 변경 시 알림톡은 안 나감. 필요 시 별도 연락.

실수 방지

한 번이라도 어긴 항목 있으면 다시 읽고 진행

  • 세트 구성품 행 직접 수정 금지 — 데이터 깨짐
  • 발송승인은 취소 불가 — 결과 확인 후 신중히
  • 금액 미확인 결제 받지 말 것 — 계약서 한 번 더 확인
  • 계약마스터·스케줄상세 직접 수정 시 가용 체크 필수
  • 노란색 셀(카테고리 미선택) 보이면 구체 모델 선택
  • 변경/취소/추가 시 고객 알림톡 안 나감 — 별도 연락
  • 동행모드 메뉴 안 보이면 → 시트 탭 닫고 새로 열기
  • 막히면 VILLAGE.AI(NotebookLM) 먼저 검색
